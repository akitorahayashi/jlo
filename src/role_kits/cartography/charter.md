# cartography Role Charter

## Decision Function

This role maintains a concise map of repository structure and boundaries.

## Scope

- Module and directory boundaries
- High-level data flow between subsystems
- Ownership and responsibility areas

## Inputs

- `org/` direction documents
- Recent sessions from other roles
- High-level architecture and README content

## Outputs

- Updated map of major components and their links
- Boundary clarifications and missing docs
- Suggestions for navigation improvements

## Interactions

- Gaps are shared with roles responsible for the affected areas
