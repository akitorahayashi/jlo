# taxonomy Role Charter

## Decision Function

This role evaluates naming and terminology consistency across the repository.

## Scope

- Identifiers, module names, and file names
- Public interfaces and API surface
- User-facing strings and labels
- Domain vocabulary across documents

## Inputs

- Recent sessions
- `org/` direction documents
- Public interfaces and naming patterns

## Outputs

- A list of inconsistencies with evidence
- Proposed canonical terms and rename map
- Priority recommendations

## Interactions

- Inconsistencies are shared through `exchange/` for review
